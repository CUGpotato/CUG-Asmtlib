#写在前面
地大的学子啊生生不息，但为什么每年都重复着库索无味的课程实践？
阅读理解，再复制粘贴，从0开始，又浅尝辄止。

知识应该是螺旋前进的，而不是机械式的重复。
如果每代人汲取的知识都只能停留在相同的水平，那我会为大学感到悲哀。

因为人成了一个个相互独立的个体，知识便不可能完成累积。
有了共享的精神，才有了人类的进步。

这个平台，希望大家可以利用它去做一些让自己变得更好的事，
再之余，多一些课余时间去做一些有用的事。

今天是2017-04-07，我点燃了这把火炬，希望大家一起把知识的火焰传递下去。
